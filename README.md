## 🔍 What is megaeth-gas?

megaeth-gas is a simple tool that helps you see the cost of gas when using MegaETH. Gas is the fee you pay to use blockchain networks like Ethereum. MegaETH uses two types of gas costs: one for compute (calculations) and one for storage (saving data). This tool shows both costs side by side.

It also compares gas fees for MegaETH against Ethereum Layer 1 (L1). You will learn how much gas you need to pay on each network. Keep in mind, the minimum gas on MegaETH is 60,000 units. Ethereum L1 is different, with a minimum gas of 21,000 units. Knowing this helps you plan better for transactions.
